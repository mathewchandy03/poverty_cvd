# The County-Level Effect of Poverty on Cardiovascular Disease Mortality


Code (you only need to run heterogeneity.R and variation_analysis.R to reproduce 
results):
cleaning.R: Data wrangling and cleaning
functions.R: Functions used in variation_analysis.R and heterogeneity.R
heterogeneity.R: Causal forest analysis and heterogeneity of treatment effects
imputation.R: Multiple imputation for left-censored data
variation_analysis.R: Decomposition of total variation into total effect and 
spurious effect
